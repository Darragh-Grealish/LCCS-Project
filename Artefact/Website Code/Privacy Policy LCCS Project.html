GDPR is the General Data Protection Regulation. It is a European regulation on data protection and privacy in the European Union and European Economic Area.

GDPR's main principles are:

  1. Lawfulness, fairness and transparency
  2. Purpose limitation
  3. Data minimisation
  4. Accuracy
  5. Storage limitation
  6. Integrity and confidentiality
  7. Accountability
  1. Processing must be lawful, fair, and transparent to the data subject.
  2. Data processed for the legitimate purposes specified to the data subject when you collected it.
  3. You should collect and process only as much data as absolutely necessary for the purposes specified.
  4. Keeping data accurate and up to date.
  5. Personally identifying data should only be stored for as long as necessary for the specified purpose.
  6. Processing is done in a way to ensure appropriate security, integrity, and confidentiality (e.g. encryption).
  7. The data controller is responsible to demonstrate GDPR compliance with all of these principles.
